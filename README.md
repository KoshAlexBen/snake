# Змейка

Простая реализация игры «Змейка» на HTML5 Canvas и JavaScript.

## Как запустить

1. Откройте файл `index.html` двойным щелчком в любом современном браузере (Chrome, Edge, Firefox и т.п.).
2. Игра загрузится автоматически.

## Управление

- Стрелки клавиатуры (`↑`, `↓`, `←`, `→`) или клавиши `W`, `A`, `S`, `D`.
- Кнопка «Заново» под полем — для перезапуска игры.

## Правила

- Собирайте еду, чтобы змейка становилась длиннее и счёт увеличивался.
- Столкновение со стеной или своим телом завершает игру.
